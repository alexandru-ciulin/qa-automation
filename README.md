QA Automation Portfolio

This repository is a learning project focused on UI test automation using Playwright with Python and Pytest.  
It demonstrates practical implementation of test automation principles and framework structure.

---

Technologies:
- Python
- Pytest
- Playwright
- Git / GitHub

---

Covered topics:
- Python fundamentals for test automation
- Unit and UI testing with Pytest
- Fixtures and test setup/teardown
- Page Object Model (POM) design pattern
- UI automation with Playwright
- End-to-end test flows (login → products → cart)
- Test structuring and separation of concerns

---

Project structure:
- pages/ → Page Object classes (LoginPage, ProductsPage, CartPage)
- tests/ → Test cases
- conftest.py → Pytest fixtures setup

---

Current features:
- Login functionality tests
- Product listing validation
- Add to cart functionality
- Cart validation
- End-to-end cart flow test

---

Status:
Work in progress (continuously improving test coverage and framework design)
